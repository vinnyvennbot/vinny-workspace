# WORKFLOWS.md - Standard Operating Procedures

## Event Planning Workflow (30-200 person events)

### **Phase 1: Initial Planning (T-30 days)**
1. **Budget confirmation** - Get per-person budget from Zed  
2. **Venue research** - Search 3-5 options, get quotes within 48h
3. **Theme development** - Define aesthetic, activities, sponsor alignment
4. **Vendor identification** - List all required services (catering, entertainment, etc.)
5. **Timeline creation** - Add all dates to Google Calendar, share with team

### **Phase 2: Vendor Procurement (T-21 days)** 
1. **Quote requests** - Send to 3-5 vendors per category, 24h follow-up rule
2. **Insurance coordination** - Send standard requirements document
3. **Competitive negotiation** - Leverage alternatives for better pricing
4. **Database updates** - Log all responses in Google Sheets immediately
5. **Financial modeling** - Create detailed cost/revenue spreadsheet

### **Phase 3: Final Execution (T-7 days)**
1. **Booking confirmations** - Zed approves all final vendors
2. **Timeline coordination** - Sync all vendor setup/breakdown times
3. **Contingency planning** - Weather backup, vendor replacement options
4. **Team briefing** - Final event overview with roles/responsibilities

## Database Management Protocol

### **Google Sheets Structure**
- **Venues/**: Capacity, pricing, availability, contact info, notes
- **Vendors/**: Service type, pricing, response time, performance rating
- **Partners/**: Sponsorship levels, brand alignment, past collaborations  
- **Influencers/**: Reach, engagement rates, partnership history

### **Update Triggers** 
1. **New vendor contact** → Add immediately with status "Contacted"  
2. **Quote received** → Update pricing, change status to "Quote Received"
3. **Booking confirmed** → Status "Confirmed", add calendar event
4. **Event completed** → Performance rating, lessons learned notes

### **Status Color Coding**
- 🟢 Green: Confirmed/Approved
- 🟡 Yellow: In Progress/Awaiting Response  
- 🟠 Orange: Needs Follow-up
- 🔴 Red: Issues/Declined
