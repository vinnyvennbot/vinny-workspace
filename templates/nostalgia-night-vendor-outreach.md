## Usage Notes

**Pre-Send Checklist:**
1. ✅ Verify event date if locked (replace [TBD] with actual date)
2. ✅ Verify venue if locked (replace "venue TBD" with actual location)
3. ✅ Use SINGLE QUOTES for --body when using gog gmail send (dollar sign protection)
4. ✅ Check RELATIONSHIPS.md for existing vendor relationships before sending
5. ✅ Search sent folder to avoid duplicate contacts: `gog gmail messages search "to:vendor@email.com"`
6. ✅ Update Mission Control VendorOutreach table IMMEDIATELY after sending

```bash
gog gmail send --to="vendor@email.com" --subject="[Subject]" --body='[Body with $prices if any]'
```
