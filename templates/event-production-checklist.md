# Event Production Checklist Template
**Purpose:** Comprehensive pre-flight checklist for all Venn Social events  
**Created:** February 21, 2026

---

## T-90 Days: Concept & Planning

### Event Design
- [ ] Event concept finalized (theme, format, experience goals)
- [ ] Target audience defined (age range, interests, size)
- [ ] Ticket pricing strategy set (early bird, standard, VIP tiers)
- [ ] Budget target established (per-person cost ceiling)
- [ ] Profitability goals confirmed (minimum margin %)
- [ ] Unique selling proposition identified (what makes this special?)

### Logistics Foundation
- [ ] Event date selected (verified day of week matches date!)
- [ ] Event time/duration confirmed (start, end, doors open)
- [ ] Capacity target set (minimum/maximum guests)
- [ ] Rain plan/backup date strategy (if outdoor)

---

## T-60 Days: Venue & Core Vendors

### Venue
- [ ] Venue researched (minimum 3 quotes per category)
- [ ] Venue selected and quote accepted
- [ ] Contract signed with venue
- [ ] Deposit paid and confirmed
- [ ] Insurance requirements confirmed
- [ ] Load-in/setup time confirmed
- [ ] Venue layout/floor plan received
- [ ] Parking/transit info documented
- [ ] ADA accessibility verified
- [ ] Bathroom count sufficient for capacity
- [ ] Noise/amplification restrictions confirmed

### Core Vendors (Entertainment, Catering, AV)
- [ ] Entertainment booked (DJ/band/performers)
- [ ] Catering selected and menu finalized
- [ ] AV/sound/lighting confirmed
- [ ] Photography/videography booked
- [ ] All vendor contracts signed
- [ ] All vendor deposits paid
- [ ] Vendor arrival times coordinated
- [ ] Load-in/setup schedule created

---

## T-45 Days: Marketing Launch

### Event Page & Ticketing
- [ ] Event page created (Luma/Eventbrite/Partiful)
- [ ] Event description written (compelling copy)
- [ ] Event photos/graphics designed
- [ ] Ticket tiers configured (early bird, standard, VIP)
- [ ] Ticket limits set (prevent overselling)
- [ ] Waitlist enabled (capture excess demand)
- [ ] Refund policy published
- [ ] FAQ section created

### Marketing Assets
- [ ] Instagram carousel designed (3-5 slides)
- [ ] Instagram story template created
- [ ] Email announcement drafted
- [ ] Social media calendar planned (pre-event posts)
- [ ] Influencer outreach list created
- [ ] Partner cross-promotion secured

### Launch
- [ ] Event page live and tested
- [ ] Tickets on sale (verify purchase flow works)
- [ ] Instagram announcement posted
- [ ] Email sent to Venn community
- [ ] Partners/sponsors notified
- [ ] Early bird pricing deadline set

---

## T-30 Days: Operations & Permits

### Permits & Insurance
- [ ] SF permits applied for (if required)
- [ ] Event insurance obtained (general liability)
- [ ] Liquor license confirmed (if serving alcohol)
- [ ] Music licensing confirmed (ASCAP/BMI if needed)
- [ ] Health permits confirmed (if serving food)

### Staffing
- [ ] Staff/volunteer roles defined
- [ ] Staff count confirmed (1:50 guest ratio minimum)
- [ ] Staff recruited and confirmed
- [ ] Staff briefing scheduled
- [ ] Staff contact list created
- [ ] Staff uniforms/identification planned

### Equipment & Rentals
- [ ] Special equipment booked (mechanical bull, photo booth, etc.)
- [ ] Tables/chairs confirmed (if not venue-provided)
- [ ] Linens/decorations confirmed
- [ ] Signage designed and ordered
- [ ] Coat check setup confirmed (if winter)

---

## T-14 Days: Final Confirmations

### Vendor Reconfirmations
- [ ] Venue reconfirmed (contact person, arrival time)
- [ ] Catering reconfirmed (final headcount, menu, timing)
- [ ] Entertainment reconfirmed (set time, technical needs)
- [ ] AV reconfirmed (equipment list, setup time)
- [ ] Photography reconfirmed (shot list, hours)
- [ ] All vendors have venue address and parking info
- [ ] All vendors have day-of contact number

### Logistics
- [ ] Final headcount submitted to venue/catering
- [ ] Seating/layout finalized
- [ ] Run-of-show timeline created (minute-by-minute)
- [ ] Staff assignments finalized
- [ ] Emergency contacts list created
- [ ] First aid kit location confirmed
- [ ] Security plan confirmed (if needed)

### Marketing Push
- [ ] "Last chance" email sent
- [ ] Instagram countdown posts started
- [ ] Waitlist converted (if space available)
- [ ] Influencer reminder posts coordinated
- [ ] Partner reminder sends

---

## T-7 Days: Pre-Event Prep

### Communications
- [ ] Attendee reminder email sent (event details, what to bring)
- [ ] Parking/transit instructions shared
- [ ] Dress code reminder (if applicable)
- [ ] Weather update sent (if outdoor)
- [ ] Contact info shared (day-of support number)

### Final Details
- [ ] All outstanding vendor payments made
- [ ] Staff briefing held (roles, timeline, emergency procedures)
- [ ] Signage printed and ready
- [ ] Name tags/wristbands prepared (if needed)
- [ ] Check-in system tested (QR codes, list, etc.)
- [ ] Backup materials prepared (extra supplies)

### Content Planning
- [ ] Shot list shared with photographer
- [ ] Instagram story plan finalized
- [ ] Live posting schedule created
- [ ] Hashtag strategy confirmed
- [ ] Social media team assigned

## T-1 Day: Final Checks

### Pre-Event Walkthrough
- [ ] Venue walkthrough completed
- [ ] Load-in plan confirmed with venue
- [ ] Setup schedule finalized
- [ ] Power/outlet locations verified
- [ ] Wi-Fi password obtained
- [ ] Emergency exits noted
- [ ] Bathroom locations confirmed

### Day-Of Prep
- [ ] All vendor arrival times reconfirmed
- [ ] Staff arrival times reconfirmed
- [ ] Weather checked (adjust if needed)

- [ ] Backup contact numbers confirmed
- [ ] Cash/payment systems tested
- [ ] Check-in materials packed

## EVENT DAY

### Pre-Event (4-6 hours before)
- [ ] Venue access confirmed (keys, codes, contact)
- [ ] Load-in completed (all vendors arrived)
- [ ] Setup completed (decorations, signage, tables)
- [ ] Sound check completed (music levels, mic test)
- [ ] Lighting tested (ambiance, safety)
- [ ] Catering setup verified (food arrived, bar stocked)
- [ ] Bathrooms stocked (TP, soap, paper towels)
- [ ] Trash cans positioned
- [ ] Check-in station ready (tablets, lists, wristbands)
- [ ] Staff briefed (final walkthrough)

### During Event (Ongoing)
- [ ] Doors open on time
- [ ] Check-in running smoothly
- [ ] Music/entertainment on schedule
- [ ] Food/beverage service flowing
- [ ] Bathrooms monitored (restocking as needed)
- [ ] Trash managed (prevent overflow)
- [ ] Social media posting (Instagram stories)
- [ ] Photography happening (candid + posed shots)
- [ ] Guest experience monitoring (address issues quickly)
- [ ] Timeline maintained (announce transitions)

### Event End (Last 30 minutes)
- [ ] Last call announced (bar/food)
- [ ] Guests thanked (announcement or personal)
- [ ] Next event promoted (if scheduled)
- [ ] Social media follow encouraged
- [ ] Guest feedback collected (optional)
- [ ] Lost & found noted

---

## POST-EVENT (Same Night)

### Breakdown
- [ ] Venue breakdown completed (decorations removed)
- [ ] Trash removed (venue left clean)
- [ ] Rentals returned (equipment packed)
- [ ] Lost & found secured
- [ ] Venue walkthrough with staff (damage check)
- [ ] Keys/access returned
- [ ] Vendors paid (outstanding balances)

### Immediate Follow-Up
- [ ] Thank you post on Instagram (event highlights)
- [ ] Best photos shared to Instagram story
- [ ] Staff thanked (personal acknowledgment)
- [ ] Vendors thanked (email + social tag if applicable)

## CRITICAL REMINDERS

### ⚠️ NEVER FORGET
1. **Verify day of week matches date** before ANY vendor communication
2. **Check event archived status** in Mission Control before vendor outreach
3. **Reply in email threads** using --reply-to-message-id (never create new threads)
4. **Use single quotes for email bodies** containing dollar signs ($)
5. **Update databases in real-time** as actions happen (not "later")
6. **Check full thread history** before responding (avoid duplicates)
7. **Get 3+ quotes minimum** per vendor category before deciding
8. **Document everything** in Mission Control immediately

### 🎯 SUCCESS METRICS
- **On-time execution** (no delays, no forgotten items)
- **Budget adherence** (within 10% of target)
- **Guest satisfaction** (positive social feedback, survey scores)
- **Vendor reliability** (all delivered as promised)
- **Profitability** (hit margin targets)
- **Replicability** (can we run this again successfully?)
