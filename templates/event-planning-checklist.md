# Event Planning Checklist - Universal Template
**Use for:** All Venn Social events (copy and customize per event)

---

## 12+ Weeks Before Event

### Concept & Approval
- [ ] Event concept finalized (theme, audience, experience)
- [ ] Financial model created (costs, revenue, profit margin)
- [ ] Zed approval obtained
- [ ] Event added to Mission Control database
- [ ] Event ID assigned (EVT-XXX)

### Venue Research
- [ ] Venue requirements defined (capacity, layout, amenities)
- [ ] 5+ venues researched
- [ ] Venue site visits scheduled
- [ ] Quotes obtained from 3+ venues
- [ ] Venue decision made
- [ ] Contract sent by venue

---

## 10-12 Weeks Before Event

### Venue & Date Lock
- [ ] **CRITICAL:** Venue contract signed
- [ ] Event date confirmed (day, time, duration)
- [ ] Venue deposit paid
- [ ] Venue insurance obtained (if required)
- [ ] Special use permits submitted (if required - Presidio, parks, etc.)
- [ ] Google Calendar event created
- [ ] Mission Control database updated with venue/date

### Vendor Outreach (Wave 1)
- [ ] Vendor categories identified (catering, entertainment, AV, etc.)
- [ ] 20+ vendors contacted per category (MANDATORY per VENDOR-OUTREACH-PROTOCOL.md)
- [ ] Vendor tracking spreadsheet created (Google Sheets)
- [ ] All outreach logged in Mission Control database
- [ ] 24-hour follow-ups sent to non-responders

---

## 8-10 Weeks Before Event

### Vendor Selection
- [ ] 5+ quotes received per category
- [ ] Quote comparison analysis completed
- [ ] Zed approval for vendor selections
- [ ] Vendor contracts signed
- [ ] Vendor deposits paid
- [ ] Payment schedule documented

### Marketing Preparation
- [ ] Event branding finalized (logo, colors, imagery)
- [ ] Marketing copy written (Eventbrite, Instagram, email)
- [ ] Event photos/graphics created
- [ ] Instagram posts drafted (announcement, countdown, day-of, recap)
- [ ] Eventbrite listing created
- [ ] Google Drive folder shared with team (zed.truong@vennapp.co)

---

## 6-8 Weeks Before Event

### Marketing Launch
- [ ] Eventbrite listing published
- [ ] Instagram announcement post (link in bio)
- [ ] Email campaign sent (if email list exists)
- [ ] Early bird ticket tier activated
- [ ] Partner/influencer outreach (comp tickets for promotion)
- [ ] Cross-promotion partnerships activated

### Logistics Planning
- [ ] Event timeline/run-of-show created
- [ ] Staff roles assigned (MC, registration, vendor liaisons, etc.)
- [ ] Equipment rental list finalized
- [ ] Load-in/load-out schedule created
- [ ] Parking/transit information documented

---

## 4-6 Weeks Before Event

### Marketing Push
- [ ] Weekly Instagram posts (countdowns, behind-the-scenes)
- [ ] Instagram Stories (daily in final 2 weeks)
- [ ] Engagement: Reply to all comments/DMs
- [ ] Ticket sales tracking (daily check-ins)
- [ ] Price tier adjustments (move from Early Bird → GA → Last Call)

### Vendor Confirmations
- [ ] All vendor contracts finalized
- [ ] Final payments scheduled
- [ ] Vendor arrival times confirmed
- [ ] Vendor setup requirements confirmed
- [ ] Vendor contact list created (day-of emergency numbers)

---

## 2-4 Weeks Before Event

### Operational Details
- [ ] Staff hired (bartenders, servers, registration, security)
- [ ] Staff briefing scheduled (1 week before event)
- [ ] Emergency contact list created
- [ ] Weather contingency plan (if outdoor)
- [ ] Insurance certificates obtained
- [ ] Alcohol permits secured (if applicable)

### Marketing: Final Push
- [ ] "Last call" marketing push (Instagram, email)
- [ ] Sold-out announcements (if applicable)
- [ ] Attendee communication (what to expect, parking, dress code)
- [ ] Pre-event engagement (polls, questions, excitement building)

---

## 1 Week Before Event

### Final Confirmations
- [ ] All vendor confirmations sent (date, time, arrival, setup)
- [ ] Venue walk-through completed
- [ ] Load-in schedule shared with all vendors
- [ ] Staff briefing completed (roles, timeline, emergency protocols)
- [ ] Day-of contact sheet printed (all vendors, staff, venue, emergency)

### Event Materials
- [ ] Signage printed (directional, welcome, branding)
- [ ] Registration materials ready (check-in list, wristbands, name tags)
- [ ] Supplies purchased (tape, markers, clipboards, etc.)
- [ ] Photography/videography scheduled
- [ ] Playlist created (if DJ not handling all music)

---

## Day Before Event

### Pre-Event Prep
- [ ] Final headcount confirmed
- [ ] Catering numbers sent to vendor
- [ ] Cash/change obtained (if cash bar/sales)
- [ ] Equipment loaded in vehicle (if self-transporting)
- [ ] Team WhatsApp/Telegram group activated
- [ ] Weather check (final contingency call)

### Communication
- [ ] Instagram Story: "Tomorrow!" reminder
- [ ] Email reminder sent to attendees (if email list exists)
- [ ] Vendor reminder texts sent (confirm arrival times)

---

## Day of Event

### Pre-Event (4-6 hours before)
- [ ] Venue access obtained
- [ ] Load-in coordination (vendors arriving on schedule)
- [ ] Setup supervision (tables, chairs, signage, decor)
- [ ] AV/lighting tested
- [ ] Registration table set up
- [ ] Bar/catering area set up

### During Event (T-minus 1 hour → end)
- [ ] Staff pre-event huddle (roles, timeline, emergency contacts)
- [ ] Final walk-through (check every detail)
- [ ] Doors open (registration ready)
- [ ] Event MC/host briefed
- [ ] Photography happening (capture key moments)
- [ ] Social media Stories posted (live event content)

### Post-Event (immediate)
- [ ] Load-out coordinated (vendors pack up)
- [ ] Venue cleaned (trash, decorations removed)
- [ ] Lost & found collected
- [ ] Final walkthrough with venue (damage check)
- [ ] Equipment returned (if rentals)
- [ ] Thank you to staff/vendors (day-of appreciation)

---

## 1-3 Days After Event

### Post-Event Admin
- [ ] Expense tracking updated (actual vs budgeted costs)
- [ ] Vendor final payments processed
- [ ] Venue final payment/deposit return processed
- [ ] Mission Control database updated (attendance, revenue, profit)

### Marketing & Engagement
- [ ] Instagram recap post (best photos, thank you message)
- [ ] Tag attendees in photos (encourage shares)
- [ ] Instagram Stories: Event highlights, behind-the-scenes
- [ ] Email thank you sent (if email list exists)
- [ ] Request feedback (Google Form or Instagram poll)

### Documentation
- [ ] Event debrief with team (what worked, what didn't)
- [ ] Vendor performance notes (for RELATIONSHIPS.md)
- [ ] Attendee feedback compiled
- [ ] Profit/loss final report
- [ ] Lessons learned documented in memory/YYYY-MM-DD.md

---

## 1-2 Weeks After Event

### Follow-Up
- [ ] Testimonials requested (for marketing future events)
- [ ] Photos/videos compiled (create highlight reel)
- [ ] Vendor thank you emails sent (maintain relationships)
- [ ] RELATIONSHIPS.md updated (vendor ratings, notes)
- [ ] Memory file created (event-specific lessons, vendor contacts)

### Future Planning
- [ ] Attendee interest survey for next event (gauge demand)
- [ ] Event series planning (if recurring event)
- [ ] Partner/sponsor recap (if applicable, plan future partnerships)

---

## Critical Reminders

**NEVER:**
- ❌ Mention budget in first vendor outreach
- ❌ Confirm bookings without signed contracts
- ❌ Contact archived event vendors
- ❌ Send emails with unverified dates (always check day of week)
- ❌ Use double quotes with dollar amounts in emails (shell escaping breaks them)
- ❌ Create new email threads when replying (always use --reply-to-message-id)
- ❌ Leave vendors on read (always acknowledge quotes/responses)
- ❌ Share Google Sheets without professional formatting
- ❌ Update anything without immediately updating database

**ALWAYS:**
- ✅ Contact 20+ vendors per category minimum
- ✅ Update Mission Control database in real-time (every action)
- ✅ Reply within email threads (maintain professional threading)
- ✅ Share all Google Drive files with zed.truong@vennapp.co
- ✅ Acknowledge vendor quotes immediately
- ✅ Verify day of week for all dates before sending emails
- ✅ Use single quotes for email bodies with prices
- ✅ Check if team members already responded before replying
- ✅ Format all shared spreadsheets professionally (colors, borders, column widths)
