# Email Signature Template - Venn Social

## Standard Format:
```
Best regards,
Vinny
Venn Social Events
vinny@vennapp.co | vennsocial.co
925-389-4794
```

## Key Requirements:
- Always include website next to email address
- Separate with pipe symbol: |
- Format: email@domain.com | website.com
- Include phone number on separate line

## Use for ALL:
- Vendor outreach
- Follow-up emails  
- Partnership proposals
- Event coordination
- Any external communications
